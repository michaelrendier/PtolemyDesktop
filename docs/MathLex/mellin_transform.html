Mellin Transform

Definition

The Mellin transform of f(x) on (0,∞) is:

M[f](s) := ∫₀^∞ x^(s−1) f(x) dx f(x) = 1/(2πi) ∫c−i∞c+i∞ x^(−s) M[f](s) ds (inverse) // s ∈ ℂ, convergence strip: a < Re(s) < b
Geometric Intuition — What It Does

Fourier transform uses additive translations (x → x+a). Mellin uses multiplicative scalings (x → λx). It's the natural transform for scale-invariant phenomena.

The substitution x = eᵘ converts Mellin to Fourier: M[f](s) = Fourier[f(eᵘ)](−is). So Mellin is Fourier on the multiplicative group (0,∞) — logarithmic frequency.

Relational Analysis — Density · Pull · Attraction

Density across scales: the Mellin transform measures how much of f lives at each multiplicative scale. The Riemann zeta function ζ(s) = M[⌊1/x⌋−1/x](s) — number-theoretic density encoded as a Mellin transform. This is the Berry-Keating connection.

Three Levels of Understanding
NOVICE High School · Powers and Scales
▶

Fourier transform asks: how much of a signal oscillates at each frequency? Mellin transform asks: how much of a signal scales at each power law? If f(x) ~ x^α for large x, that shows up as a pole of M[f](s) at s = −α.

JOURNEYMAN Collegiate · Gamma Function
▶

The Mellin transform of e^(−x) is the Gamma function: Γ(s) = M[e^(−x)](s). The functional equation ζ(s) = 2^s π^(s−1) sin(πs/2) Γ(1−s) ζ(1−s) is a Mellin symmetry statement.

MASTER Graduate / PhD · Perron's Formula
▶

For arithmetic functions a(n), the Dirichlet series Σ a(n)n^(−s) is the Mellin transform of Σ a(n)δ(x−n). Perron's formula inverts: Σn≤x a(n) = (1/2πi)∫ [Σa(n)n^(−s)] x^s ds/s. The Riemann Hypothesis is equivalent to the optimal error bound in this inversion.

Derivation
// Mellin of x^a·e^(-x) M[x^a · e^(-x)](s) = ∫₀^∞ x^(s-1) · x^a · e^(-x) dx = ∫₀^∞ x^(s+a−1) e^(−x) dx = Γ(s+a) Special case a=0: M[e^(−x)](s) = Γ(s) // the Gamma function is a Mellin transform

Common Mistakes
MISTAKE 01
Confusing Mellin with Laplace. Mellin is ∫₀^∞ x^(s−1) f(x) dx; Laplace is ∫₀^∞ e^(−sx) f(x) dx. Related by x = e^(−t).
MISTAKE 02
Ignoring the strip of convergence. M[f](s) may only converge for a < Re(s) < b.
MISTAKE 03
Expecting the inverse to be real when f is real — it requires a complex contour integral.
