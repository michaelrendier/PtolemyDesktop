Chebyshev Polynomials

Optimal polynomial approximation — minimax error on [−1,1]
DEFINITION
TRIGONOMETRIC DEFINITION
T_n(cos θ) = cos(nθ)
RECURRENCE
T_0 = 1, T_1 = x
T_{n+1}(x) = 2x · T_n(x) − T_{n−1}(x)

First polynomials: T₀=1, T₁=x, T₂=2x²−1, T₃=4x³−3x, T₄=8x⁴−8x²+1

MINIMAX PROPERTY

Among all polynomials of degree n with leading coefficient 1, T_n(x)/2^{n−1} has the smallest maximum deviation from zero on [−1,1].

MAX ERROR BOUND
‖f − p_n‖ ≤ max|f^{(n+1)}(x)| / (2^n · (n+1)!)
RADIAN FORM — ROOTS AND NODES
ROOTS (CHEBYSHEV NODES)
x_k = cos((2k−1)π / 2n), k = 1,...,n
Optimal interpolation nodes — minimize Runge's phenomenon
PTOLEMY CONNECTION
LORENZ-STIRLING / OUTPUT ENGINE

Chebyshev polynomials are the basis for the spectral representation of the Lorenz-Stirling attractor system. The Newton basin routing in the Output Engine uses Chebyshev node placement to achieve stable numerical convergence on the degree-9 Stirling polynomial. Chebyshev nodes avoid the edge clustering that would cause the output distribution to pile up at extreme semantic values.
