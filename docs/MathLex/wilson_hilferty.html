Wilson-Hilferty Approximation

Cube root transformation — normalizing the chi-squared distribution
DEFINITION

The Wilson-Hilferty approximation converts a chi-squared random variable to approximately normal using a cube root transformation.

WILSON-HILFERTY TRANSFORM
X ~ χ²(k) ⟹ (X/k)^{1/3} ≈ N(1 − 2/(9k), 2/(9k))
STANDARDIZED FORM
Z = ((X/k)^{1/3} − (1 − 2/(9k))) / √(2/(9k)) ≈ N(0,1)
WHY CUBE ROOT

The cube root is the variance-stabilizing transformation for the chi-squared family. It removes the heavy right tail, making the distribution symmetric enough for normal approximation.

Accuracy: excellent for k ≥ 3. At k=1, the approximation is poor — the chi-squared is essentially half-normal.
ERROR BOUND
ABSOLUTE ERROR IN CDF
|F_χ²(x) − Φ(Z)| ≤ C / k
where C is a bounded constant, Φ is the standard normal CDF
