Minkowski Metric

ds²=−dt²+dx²+dy²+dz² · flat spacetime
MINKOWSKI METRIC — GEOMETRIC VISUALIZATION

Definition

The Minkowski metric on ℝ1,3 (flat spacetime):

ds² = ημνdxμdxν = −c²dt² + dx² + dy² + dz²η = diag(−1, +1, +1, +1) (signature −+++)// Invariant: ds² same in all inertial frames
Geometric Intuition — What It Does

The minus sign on dt² is everything. It creates the light cone: ds²=0 for light-like paths. ds²<0 for time-like paths (massive objects). ds²>0 for space-like separations (inaccessible).

The Minkowski metric is not positive-definite — it's pseudo-Riemannian. Distances can be zero (null) or negative (time-like). The invariant interval is proper time.

Relational Analysis — Density · Pull · Attraction

Causal density: the light cone at each event divides spacetime into three density regions — future (accessible), past (influencing), and spacelike (unreachable). The metric defines which events can attract/influence each other. Causal structure = the geometry of mutual attraction in spacetime.

Three Levels of Understanding
NOVICE High School · The Minus Sign Changes Everything
▶

Euclidean distance: ds² = dx² + dy². Always positive. Minkowski: ds² = −dt² + dx². Can be positive, zero, or negative. When ds²=0, you're on the light cone — moving at the speed of light. When ds²<0, the temporal part dominates — you're on a timelike path through time.

JOURNEYMAN Collegiate · Raising and Lowering Indices
▶

ημν and its inverse ημν = diag(−1,+1,+1,+1) lower and raise indices: Vμ = ημνVν. For a 4-vector V=(V⁰,V¹,V²,V³): V0=−V⁰, Vi=+Vⁱ. The dot product V·W = VμWμ = −V⁰W⁰ + V^iW_i is Lorentz invariant.

MASTER Graduate / PhD · Penrose Diagram
▶

Conformal compactification of Minkowski space: apply r = tan(R), t = tan(T+R)/2 + tan(T−R)/2 type maps to bring infinity to finite position. The Penrose diagram is a finite square encoding the full causal structure. Past/future null infinity (𝒥±), spacelike infinity (i⁰), and timelike infinity (i±) all appear as boundaries.

Common Mistakes
MISTAKE 01
Using (+,−,−,−) when the rest of your work uses (−,+,+,+). Pick a convention and stick to it.
MISTAKE 02
Treating ds²<0 as an error. Time-like intervals have negative ds² — that's the correct signature.
MISTAKE 03
Confusing proper time τ with coordinate time t. They agree only for a particle at rest.
