Catalan Numbers

Counting binary trees, bracket sequences, and polygon triangulations
DEFINITION
CLOSED FORM
C_n = (1/(n+1)) · C(2n, n) = (2n)! / ((n+1)! · n!)
RECURRENCE
C_0 = 1, C_{n+1} = Σ_{i=0}^{n} C_i · C_{n−i}

First values: 1, 1, 2, 5, 14, 42, 132, 429, 1430, 4862...

WHAT THEY COUNT

GENERATING FUNCTION
ORDINARY GF
C(x) = (1 − √(1−4x)) / 2x
ASYMPTOTIC (RADIAN)
C_n ~ 4^n / (√π · n^(3/2))
PTOLEMY CONNECTION
HYPERWEBSTER / LSH_DATATYPE

Catalan numbers count the number of distinct parse trees for a string of n tokens — directly relevant to GrammarNeuron's parse space. The Horner bijection nests evaluations in a binary tree structure; the number of distinct nestings of a k-dimensional address is C_{k−1}. This is also the count of distinct ways to fold a SemanticWord's 12-layer stack into evaluation order.
