Definition

Cayley-Dickson construction is a recursive doubling operation on a *-algebra that produces a new algebra of twice the dimension. Given an algebra A with conjugation a*, it constructs A' whose elements are ordered pairs (a, b) with a, b ∈ A.

multiplication: (a, b)(c, d) := (ac − d*b, da + bc*) conjugation: (a, b)* := (a*, −b) // each doubling costs one algebraic property
ℝ → ℂ (dim 2, gains: complex rotation, loses: ordering) ℂ → ℍ (dim 4, gains: 3D rotation, loses: commutativity) ℍ → 𝕆 (dim 8, gains: 7-axis rotation, loses: associativity) 𝕆 → 𝕊 (dim 16, loses: alternativity — division breaks)
Geometric Intuition — What It Does

Each doubling adds a new independent axis of rotation. Think of it as nesting gyroscopes:

  • ℂ — one axis. i rotates the plane 90°. A single spinner.
  • ℍ — three axes. j, k nest inside i. A gimbal. Quaternions describe 3D orientation with no gimbal lock.
  • 𝕆 — seven axes. e₁e₂e₃e₄e₅e₆e₇. A hyper-gyroscope. The 7 imaginary units interact in ways that create pull — each axis attracts related axes.

The loss of associativity in 𝕆 means order of rotation matters absolutely. This is why octonions model attention: the path to a result changes the result. Context is geometric.

Relational Analysis — Density · Pull · Attraction

Density appears here as dimensional concentration. The norm of an octonion:

‖x‖² = x₀² + x₁² + ··· + x₇² // 8 scalar components competing for the same unit sphere // concentration of norm = density of meaning in that direction

Attraction is the multiplication table itself. e₁e₂ = e₄ — these axes pull toward each other to produce a third. The Fano plane encodes which axes attract which. This is structurally identical to how attention weights in a neural network determine which tokens pull toward which outputs.

Same density, different notations: ‖x‖², probability mass, attention weight, gravitational potential, information entropy — all measure how much is concentrated where.

Three Levels of Understanding
NOVICE High School · The Doubling Idea
▶

You know real numbers. Square root of −1 doesn't exist in reals, so mathematicians invented i and called the result complex numbers ℂ.

Cayley-Dickson says: do that again. Take pairs of complex numbers. Define multiplication carefully. You get quaternions ℍ — four numbers that describe 3D rotations perfectly. Do it again: octonions 𝕆, eight numbers.

Each time you double, you gain new rotational power but lose one rule you're used to. Octonions don't follow (ab)c = a(bc). Order matters.

JOURNEYMAN Collegiate · *-Algebra Construction
▶

Given a *-algebra A over ℝ with conjugation a ↦ a*, norm N(a) = aa*, the Cayley-Dickson double is A² = A × A with:

(a,b)(c,d) := (ac−d*b, da+bc*) (a,b)* := (a*, −b) N(a,b) = N(a) + N(b)

Key property: composition algebras — the norm is multiplicative: N(xy) = N(x)N(y). This holds for ℝ, ℂ, ℍ, 𝕆 only. Hurwitz's theorem proves these are the only normed division algebras. At 𝕆 the construction stops being a division algebra if continued.

MASTER Graduate / PhD · Exceptional Structures
▶

The Cayley-Dickson algebras are classified by Hurwitz's theorem: only ℝ, ℂ, ℍ, 𝕆 admit a multiplicative quadratic form (composition algebras). The proof uses the theory of Clifford algebras and Radon-Hurwitz numbers.

𝕆 is the unique exceptional Jordan algebra root — the Albert algebra J(3,𝕆) is 27-dimensional and tied to E₈ root system and string theory's 10 = 8+2 dimensions via triality.

Non-associativity of 𝕆 means left/right multiplication operators L_x, R_x generate the exceptional Lie algebra G₂ as the automorphism group of 𝕆. This is why G₂ appears in gauge theory and why the Monster group has octonion fingerprints.

Aut(𝕆) ≅ G₂ (14-dimensional exceptional Lie group) G₂ ⊂ Spin(7) ⊂ SO(8) (triality chain) // triality = the 3-way symmetry between 𝕆's three 8-dim representations
Derivation — Building the Multiplication Rule
// Goal: define multiplication on pairs (a,b) that preserves the norm // Require: N(xy) = N(x)N(y), where N(a,b) = N(a)+N(b) Step 1 — expand N(xy): N((a,b)(c,d)) = N(ac−d*b) + N(da+bc*) Step 2 — expand each term using N(x+y) = N(x)+N(y)+2Re(xy*): N(ac−d*b) = N(a)N(c) + N(d)N(b) − 2Re(ac(d*b)*) N(da+bc*) = N(d)N(a) + N(b)N(c) + 2Re(da(bc*)*) Step 3 — cross terms cancel when Re terms are conjugates: ∴ N((a,b)(c,d)) = (N(a)+N(b))(N(c)+N(d)) ∴ N((a,b)(c,d)) = N(a,b) · N(c,d) ✓ // This uniquely determines the multiplication formula above

Common Mistakes
MISTAKE 01
Assuming quaternion multiplication is commutative. pq ≠ qp in general. Only the scalar part commutes.
MISTAKE 02
Applying associativity to octonions. (e₁e₂)e₄ ≠ e₁(e₂e₄). Always check the Fano plane multiplication table.
MISTAKE 03
Confusing the doubling formula variants. There are two sign conventions in the literature. Stick to one. This document uses (ac−d*b, da+bc*).
MISTAKE 04
Continuing the construction past 𝕆 and expecting useful algebra. Sedenions (𝕊, dim 16) have zero divisors — non-zero elements that multiply to zero. Division breaks completely.
