PTOLEMY PROJECT ARCHITECTURE
════════════════════════════════════════════════════════════════════════════════

  Ptolemy is a large-scale personal software system organized as named Faces,
  each corresponding to a historical figure from the Library of Alexandria.

  Face          Purpose
  ────────────────────────────────────────────────────────────────────────────
  Alexandria    OpenGL visualization experiments (The Grid process monitor,
                city renderer, 3D graphing calculator)
  Archimedes    Mathematics and science
  Callimachus   Archival and database operations — will eventually absorb
                HyperWebster indexing, phasing out SQL dependency
  Anaximander   Navigation and travel — hot springs, libraries by location
  Kryptos       Encryption — Caesar/Vigenère/Enigma (Code Book program)
                May become subset of Callimachus
  Phaleron      Internal search and presentation algorithm
  Pharos        Core functions — the Lighthouse
  Tesla         Device interfacing — OBD2, VAG-COM bluetooth
  Mouseion      Flask website — thewanderinggod.tech — Theater, Art Gallery,
                Music Studios, content display
  Ptolemy++     C++ port for performance-critical components (PyQt5/QGraphicsScene)
  ────────────────────────────────────────────────────────────────────────────

  Jupyter notebooks: excluded from Ptolemy architecture entirely.
                     Personal experimentation only.

MONAD NEURAL ARCHITECTURE (ptolemy_core.py)
════════════════════════════════════════════════════════════════════════════════

  Six-layer stack (bottom to top):
    Layer 0: CharacterNeuron — file-type recognition, alphabet identification
    Layer 1: LexicalDimensionNeuron
    Layer 2: GrammarNeuron — language-specific variation
    Layer 3: WordMonadNetwork — monad isolation (anti-catastrophic-forgetting)
    Layer 4: WernickeNetwork — comprehension
    Layer 5: BrocaNetwork — production

  The "it" pronoun problem: structurally resolved via Monad architecture
  Monad isolation: each word is a self-contained unit — prevents forgetting
  Cross-language bridges: meaning vectors at GrammarNeuron layer
  Training strategy: EWC (Elastic Weight Consolidation)
  Julius Caesar face configuration: defined, not yet trained
