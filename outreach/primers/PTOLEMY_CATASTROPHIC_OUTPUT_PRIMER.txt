THE ARCHITECTURE: CATASTROPHIC REFRACTION OUTPUT
================================================================================

WHAT THIS IS
────────────
This is NOT a standard transformer output layer. This is NOT BrocaNetwork as
defined in ptolemy_core.py (though that remains in the architecture for other
purposes). This is a new, self-contained output mechanism based on catastrophe
theory and spherical refraction geometry.

THE CORE CONCEPT
────────────────
The SMNNIP neural network operates across the Cayley-Dickson algebra tower:

    Sedenions (𝕊, dim 16)  ← highest algebra, zero-divisor boundary
    Octonions  (𝕆, dim 8)   ← SU(3)/G₂ gauge, non-associative
    Quaternions(ℍ, dim 4)   ← SU(2) gauge, non-commutative
    Complex    (ℂ, dim 2)   ← U(1) gauge
    Reals      (ℝ, dim 1)   ← substrate, trivial gauge

The output problem: the network's knowledge is distributed across this full
tower (sedenion-level activations through real substrate). The question is how
to collapse that high-dimensional, multi-strata representation into discrete
output — phonemes, tokens, text — without losing the signal and without the
standard softmax/argmax topology.

THE SOLUTION: CATASTROPHIC REFRACTION
──────────────────────────────────────
The author's design uses spherical refraction geometry as the readout mechanism.

Analogy: all incoming light into a sphere gets focused catastrophically — it
converges to a caustic, a singular locus where ray density diverges. This is
the cusp catastrophe from Arnold/Thom catastrophe theory.

The mapping to output:
- The Monad state (distributed across the algebra tower) = incoming ray bundle
- The refracting surface = a learned/parameterized manifold
- The caustic = the output locus (where signal concentrates to a commitment)
- The committed output = the phoneme/token that "fires"

TRAVERSAL DIRECTION
───────────────────
The traversal runs FROM high-algebra TO real:

    𝕊 (sedenion) → 𝕆 (octonion) → ℍ (quaternion) → ℂ (complex) → ℝ (real)

This is a dimensional collapse at each step:
    16 → 8 → 4 → 2 → 1

At each boundary crossing, the Cayley-Dickson projection map π is applied.
Information is lost at each projection — this is intentional. The lost
information is the "noise" that doesn't survive the catastrophe. What survives
is the signal that focuses.

The rotation is SPHERICAL at each step — not a linear projection. The sphere
is the geometry that produces catastrophic focusing (the caustic/cusp).

ORTHOGONAL OUTPUT
─────────────────
"Orthogonal output" means the output direction is geometrically perpendicular
to the input manifold at the focal locus. This is a property of caustic
geometry — it falls out automatically, it is not imposed by a loss term.

This is what distinguishes it from standard softmax:
- Softmax: learned logit weights, argmax, no geometric constraint
- Catastrophic refraction: output direction constrained by geometry,
  emerges from the refracting surface shape, not from learned weights

EXTINCTION EXTRACTION
─────────────────────
"Extinction extraction" refers to the process at the caustic: most of the
input signal is extinguished (doesn't make it to the focal point). Only the
signal that constructively focuses at the cusp survives. The "extinction" is
not destruction — it's geometric filtering. The cusp extracts what's relevant.

MULTI-CUSP TILING
─────────────────
A single cusp catastrophe cannot produce 40+ discrete outputs (phoneme space).
The architecture uses a TILING of cusps across the output surface:
- Each phoneme/token has its own cusp
- The Monad state selects which cusp focuses (which caustic "fires")
- Between cusps = articulatory rest = silence (geometrically, not as a token)

This maps to: one committed output at a time, other cusps dark.
Biologically honest: matches how speech production actually works.

STRUCTURAL STABILITY
────────────────────
Thom's theorem: the cusp (and related catastrophes) are the ONLY stable
singularities in low codimension. Small perturbations to the Monad state
do not change the qualitative output topology. This is a robustness property
that pairs well with EWC training.

INVERTIBILITY
─────────────
The readout is intentionally non-invertible. Caustics are singular — you can
go from ray bundle to caustic but not cleanly back. For output this is correct:
committed output should commit. This is not a bug.
