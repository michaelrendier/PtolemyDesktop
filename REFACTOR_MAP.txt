INFERENCE PATH (verified v0.9)
══════════════════════════════════════════════════════════════════════════════

INPUT:
  text → PtolShell._ptolemy_command() → PtolShell._ptolemy_respond()

THREAD:
  _SMNNIPThread → PtolBrain.generate(command)

BRAIN:
  PtolBrain.__init__:
    load weights from Philadelphos/weights/*.pkl
    restore SMNNIPLayer3 (𝕆 octonion layer)
    optional: _ReverseTower (Noether current annotation)
    optional: lorenz_stirling (basin routing annotation)

  PtolBrain.generate(command):
    _ReverseTower → Noether current → annotate context
    seed context from command tail
    L3.forward() → octonion logits → temperature sampling → char
    yield char  (streamed to UI via pyqtSignal)
    at sentence boundary → LorenzStirling basin annotation → stop

FILTER:
  PtolemyTongue.filter(text)
    fold-count heuristic → pentagonal / hexagonal geometry check
    rejects pathological glyphs (runaway repeat, punct storm, etc.)

OUTPUT:
  write_fn(filtered_text, '#aaffcc')  → _FallbackTerm.write() or QTermWidget
