The following colouring method doesn't have a helpfile.


fBm Gauss Colouring Method
--------------------------------

This method is in the mde.ucl folder for the more French version,  
mde-eng.ucl for the English version and mde3.ucl for the version with all the textures and automask.

This colouring method is a personal version of Kerry 
Mitchell's "Gaussian Integer".

I would like to thank Kerry for his permission to use it.

I just took off the 'integer type' parameter of the original method and
changed it for a parameter function. So, any function available as parameter
in UF can be used instead of just the integer ones.

For the other parameters, look in mde-toolbox.htm for the appropriate toolboxes.
