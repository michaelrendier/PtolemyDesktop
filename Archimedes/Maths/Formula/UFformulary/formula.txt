ULTRA FRACTAL FORMULA COMPILATION
Release 08 Apr 2000


*** What is this? ***

This is a compilation of publicly-available formulas for Ultra Fractal
version 2.  (Ultra Fractal is a shareware fractal generator for Windows
95 and NT.  See www.ultrafractal.com for more information.)  This
compilation is composed of calculation formulas (.ufm files), transformation
formulas (.uxf files), and coloring formulas (.ucl files).  The formulas
have been written by several authors (see registry below), and this
collection draws both from the authors' own collections and from postings
to the UltraFractal mailing list.

This collection does not include the Fractint formulas that have been
assembled in George Martin's Orgform collection.  Nor does this collection
include the UltraFractal "Standard" formulas or the Fractint compatibility
files.  Those are included in the UltraFractal distribution.

This compilation is maintained by Kerry Mitchell (lkmitch@inficad.com).


*** How to use the compilation ***

After unzipping, simply put all the .ufm, .uxf, and .ucl files into the
appropriate directories as indicated by your UF options settings
(see Options|Options|Directories in UltraFractal).

Some authors are now including help files.  If there is a file like
"xxx-help.zip", see the "readme" file in the zip archive for information
on how to use the helpfiles.

To make older parameter collections compatible with these formulas, some
name changes may need to be made to the parameter (.upr) files.  Most can
be made by removing all instances of "-pub" (without quotes).  For example,
change "dmj-pub" to "dmj" by removing "-pub".
